16マス表示するメモ

・まずマスを ■ 記号とする。
座標を用意して左上を初期位置と設定して0,0とすると
右下が3,3、つまり0,0～3,3までの要素を持った配列を作成
↓
各々

・プレイヤーに操作したいマスの番号を入力させ、
　それと値が等しい配列の要素のマスを■から□にする

・for文の中でcout << □ << flushなどとして
　なんとかそれぞれの□に数字を記録しておいて
　□に記録した数字と配列の数字を照らし合わせて
　入力値と一致する□を■に変えられないか

・1から16のマスをメンバ変数にすれば、コンストラクタで
　初期化でき、オブジェクトごとにマスを持つことができる

　cin >> input_num;
　for(int i = 0; i < 5; ++i) {
    for(int j = 0; j < 5; ++j) {
        if(input_num == masu[i][j]) {
            masu[i][j]のマスを□に変更
        }
    }
}

疑問点や提案
・配列の各要素と座標を結びつけるにはどうすればよいか？
・16個の配列ではなく、4この配列と4個の配列で
　x座標とy座標をわけたほうがよいのではないか

進捗
・16個の数字を4 * 4の形で表示する　✓
・数字ではなく□で表示する
・プレイヤーが入力した座標のマスを□から■に変換する
