HTML タグ一覧

<img src="" alt="テスト画像">
　→ ページに画像を挿入する。alt属性は画像を観られない人に向けて説明するテキストを設定するもの。
　　imgタグは空要素のため、子要素や終了タグは置けない。
　　alt属性は画像の説明だが、これは例えば、画像のURLが間違っている時にwebページ上ではaltが表示されるので
　　パスが間違っていることが分かりやすい。

<head>
　→ ページの閲覧者に向けて表示するためのコンテンツではない、HTMLページに含めたいものを全て納めるための
　　入れ物。検索エンジン向けのキーワードやページのディスクリプション(説明書き)、CSS、文字コードの宣言など

<a>
　→ リンクを追加する。href属性にリンク先になるwebアドレスを書き込む。
　　target属性にblankを指定すると新しいタブでリンク先を開く。
　　title属性はカーソルをURLに合わせた時に表示するタイトル。
　　ブロックレベル要素を含む、ほぼすべてのコンテンツをリンクにすることが可能。(見出しなど)
　　download属性は、ブラウザで開くのではなくクリックするとダウンロードができるリソースを指定する。

<script>
　→ JavaScriptをページに適用する。

<input>
　→ 入力フォームを配置する。type属性で入力値の形を指定する？
　　disabled属性をつけるとグレーアウトし、入力を受け付けなくなる。

<meta>
　→ データを説明するデータ。文字コードやページの著者、ページのdescriptionなどを設定できる。

<span>
　→ 何の意味も持たない。CSSを適用、またはJavaScriptで何かをするときにコンテンツを囲むために使用する。

<ul>
　→ すべての順序なしリストはこのタグで開始する。

<li>
　→ リスト。

<ol>
　→ 番号付きリスト。

<em>
　→ 強調。

<strong>
　→ 太字。

<dl>
　→ 説明リスト。

<dt>
　→ <dl>で囲まれた「用語」を囲む。

<dd>
　→ <dl>で囲まれた「用語の説明」を囲む。

<sup>
　→ 上付き文字にする。xの2乗の「2」などを上に持ってくることができる。

<sub>
　→ 下付き文字にする。化学式の数字などを右下に持ってくることができる。

<pre>
　→ 空白を保持する。通常、HTMLテキストの中で空白を使用するとブラウザでは無視されるが、
　　このタグで囲むと同じように空白が表示されるようになる。

<time>
　→ 時間と日付を機械可読形式でマークアップする。

<br>
　→ 改行。

<canvas>
　→ JavaScriptで2Dグラフィックを描画するAPIを提供する。

<video>
　→ 動画を埋め込む。src属性で動画のパスを指定。controls属性で動画や音声の再生を制御できる。

<audio>
　→ 音声を埋め込む。

<iframe>
　→ 他のWebページなどを埋め込むことができる？

<sandbox>
　→ 害を発生させることができないなんかすごいやつ。常に引数なしのsandboxを使用すれば大丈夫そう。
　　サンドボックス化されていないコンテンツは、JavaScriptの実行、フォームの送信、
　　ポップアップウィンドウの起動などを行うことができてしまうため、これを指定するのが良い。

<table>
　→ テーブルを作成する。

<th>
　→ テーブルの見出しを指定する。

<tr>
　→ 列を指定していく。

<td>
　→ 行(要素)を指定していく。

<caption>
　→ キャプションをつける。
