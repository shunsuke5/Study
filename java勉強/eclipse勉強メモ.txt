eclipse 勉強メモ

・何かを実行したときにいちいちショートカットキーが表示されるのは「キーバインド」と言う。
　これを非表示にしたい場合、「ウィンドウ」→「設定」→「一般」→「キー」の、
　「コマンドが呼び出されたときにキー・バインドを表示する」のチェックを外す。

=================================================================================================================================

ショートカットキー

ctrl + N
　→ 新規作成(作成したい場所を選択するとその直下に作成)

ctrl + W
　→ ファイルを閉じる

ctrl + shift + W
　→ ファイルをすべて閉じる

F2
　→ ファイル名変更

ctrl + shift + S
　→ ファイルをすべて保存

ctrl + D
